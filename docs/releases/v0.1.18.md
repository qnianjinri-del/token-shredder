# Token Shredder v0.1.18

A tiny local desktop pet that shreds your AI token spending in real time.

This update adds a standalone online demo page for people who want to understand the project before installing the macOS app.

## Highlights

- Added `docs/demo.html`.
- The demo page shows:
  - animated pet preview
  - first-run usage paths
  - `POST /usage` curl example
  - troubleshooting visual
  - local-first privacy boundary
- Linked the demo from the GitHub Pages landing page.
- Linked the demo from README and Launch Kit.
- Added `demo.html` to `docs/sitemap.xml`.
- Verified the demo page with a Chrome headless screenshot.
- Hardened macOS packaged smoke testing for local workspaces with non-ASCII paths and kept the v0.1.x mac build as an unsigned local distribution.

## Try It

Open:

```txt
https://qnianjinri-del.github.io/token-shredder/demo.html
```

The online demo is only a visual explanation. Real usage monitoring still requires the local Electron app.

## Privacy Notes

- The demo page does not collect data.
- It does not call a backend.
- It does not read local prompts, completions, usage logs, API keys, or messages.
- It exists only to explain the local desktop app experience.

## Known Limits

- The demo page is not the actual desktop app.
- The macOS build remains unsigned and not notarized.
- macOS helper processes may still appear as Electron helper processes in Activity Monitor for this unsigned v0.1.x build.
- A real recorded desktop GIF / MP4 would still be better for social posts.
