# Token Shredder v0.1.2

A tiny local desktop pet that shreds your AI token spending in real time.

This update focuses on first-run friction. People can now try the app without any API key, then connect real usage when they are ready.

## Highlights

- Added a one-click local quick demo for first-time users.
- The quick demo creates a clearly labeled local simulated usage event, drives the pet once, then stops on the resulting cost progress.
- Added a new original pixel skin: Budget Black Hole.
- Token Shredder now ships with 6 switchable desktop pet skins.
- Updated onboarding copy to explain the two real usage paths:
  - local OpenAI-compatible proxy
  - direct `POST /usage`
- Refreshed the skin showcase and social preview assets.

## How To Try It

Download the latest macOS build from GitHub Releases.

To build from source:

```bash
npm install
npm run dev:desktop
```

On first launch, open backstage and click `先不填 Key，试玩一下`. No provider key is required for that demo.

## Local API Example

```bash
curl -X POST http://127.0.0.1:17391/usage \
  -H "Content-Type: application/json" \
  -d '{"source":"my-agent","scenarioName":"repo cleanup","inputTokens":120000,"outputTokens":45000}'
```

## Privacy Notes

- Runs locally on your machine.
- Collector and proxy listen on `127.0.0.1`.
- Records token counts, source names, scenario names, timestamps, and local cost estimates.
- Does not need to record prompts or completions.
- Does not upload usage data.
- Does not use third-party analytics.

## Known Limits

- macOS is still the primary target.
- Builds are unsigned and not notarized.
- Windows and Linux need community testing.
- Full streaming usage extraction is not implemented.
- SDK / CLI wrappers are not implemented yet.
- The quick demo is simulated local usage and is not real billing.
