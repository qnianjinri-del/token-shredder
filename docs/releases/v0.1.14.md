# Token Shredder v0.1.14

A tiny local desktop pet that shreds your AI token spending in real time.

This update focuses on making the backstage window less overwhelming. The pet is still the main product surface; the backstage is now organized around what a new user should do next, how to connect usage, how to inspect cost, how to tune the pet, and how to diagnose problems.

## Highlights

- Reorganized backstage into five tabs: `开始 / 接入 / 成本 / 宠物 / 诊断`.
- Kept first-run actions in `开始`: `下一步建议`, `自动体检`, `从这里开始`, and 30-second verification.
- Grouped provider setup, integration recipes, and usage monitoring under `接入`.
- Grouped pricing, cost breakdown, sharing, and session export under `成本`.
- Grouped skin selection, pet size, and demo mode under `宠物`.
- Grouped runtime status, diagnostics, backup / restore, and reset under `诊断`.
- Added a plain-language next action to automatic self-checks and copied reports.
- Added tests for self-check next-action selection.
- Added a pull request template with validation and local-first privacy checklists.
- Added a privacy / security review issue template.
- Updated README, Chinese docs, roadmap, release guide, and the launch page.

## How To Try It

1. Open Token Shredder.
2. Right-click the desktop pet and choose `进入后台`.
3. Start from the `开始` tab.
4. Click `运行自动体检`.
5. Follow the displayed next action, or click `一键试玩` to see the pet move without an API key.

## Privacy Notes

- No cloud backend or analytics were added.
- The self-check report intentionally excludes API keys, Authorization headers, prompts, completions, messages, and request bodies.
- Usage collection remains local-first on `127.0.0.1`.
- Pricing remains user-configured and is not official live provider pricing.

## Known Limits

- macOS remains the primary target.
- Builds are unsigned and not notarized.
- The local proxy remains basic.
- Streaming usage extraction remains limited.
- Windows / Linux still need community testing.
