# Token Shredder v0.1.3

A tiny local desktop pet that shreds your AI token spending in real time.

This update focuses on making the first real verification path clearer. The goal is simple: a new user should be able to open the app, click the quick demo, then send a real local usage event without reading the whole README.

## Highlights

- Added a backstage `30 秒验证` panel.
- The verification panel includes:
  - one-click no-key demo
  - local collector smoke test
  - copy-paste example commands
- Added runnable examples:
  - `examples/post-usage-node.mjs`
  - `examples/post-openai-style-usage.mjs`
  - `examples/post-usage-python.py`
- Updated README quick start with a dedicated 30-second verification path.
- Updated the GitHub Pages launch page to show the quick verification flow.

## How To Try It

Download the latest macOS build from GitHub Releases, or build from source:

```bash
npm install
npm run dev:desktop
```

Downloaded app users can use the backstage `30 秒验证` card. If you cloned the repository, you can also run:

```bash
node examples/post-usage-node.mjs
python3 examples/post-usage-python.py
node examples/post-openai-style-usage.mjs
```

If the app uses another local port, set:

```bash
export TOKEN_SHREDDER_URL="http://127.0.0.1:17392/usage"
```

## Privacy Notes

- Runs locally on your machine.
- Collector and proxy listen on `127.0.0.1`.
- Examples send only usage numbers.
- Examples do not send prompts, completions, messages, or API keys.
- Token Shredder does not upload usage data.

## Known Limits

- macOS is still the primary target.
- Builds are unsigned and not notarized.
- Windows and Linux need community testing.
- Full streaming usage extraction is not implemented.
- SDK / CLI wrappers are not implemented yet.
