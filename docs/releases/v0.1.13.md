# Token Shredder v0.1.13

A tiny local desktop pet that shreds your AI token spending in real time.

This update adds a one-click backstage self-check. The goal is to make support and first-run debugging less mysterious: users can run one check, copy one report, and see whether localhost, pricing, provider fields, collector tests, and real usage are in a good state.

## Highlights

- Added `自动体检` to the backstage window.
- The self-check covers:
  - local collector state
  - `GET /health`
  - `POST /usage` collector test
  - setup-readiness status
  - editable pricing status
  - provider required fields
  - real usage vs demo/test usage
  - pet runtime state
- Added copyable self-check reports for issues or agent handoff.
- Self-check reports intentionally exclude API keys, Authorization headers, prompts, completions, and messages.
- Added reusable self-check helpers and unit tests.
- Updated README, Chinese docs, release guide, roadmap, and launch page copy.

## How To Try It

1. Open Token Shredder.
2. Right-click the desktop pet and choose `进入后台`.
3. Click `运行自动体检`.
4. Click `复制体检报告` if something looks wrong.

## Privacy Notes

- The report only includes local status, local endpoints, setup state, and non-secret configuration status.
- It does not include API keys.
- It does not include prompt, completion, message, or Authorization header contents.
- No cloud backend or analytics were added.

## Known Limits

- macOS remains the primary target.
- Builds are unsigned and not notarized.
- The local proxy remains basic.
- Streaming usage extraction remains limited.
