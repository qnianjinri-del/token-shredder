# Token Shredder v0.1.4

A tiny local desktop pet that shreds your AI token spending in real time.

This update focuses on sharing. Once the pet has shredded a session, it should be easier to explain the project to someone else without writing the post from scratch.

## Highlights

- Added English and Chinese launch-post copy buttons to the backstage share panel.
- Share text includes:
  - session cost
  - destroyed bill count
  - current bill progress
  - local-first / no prompt logging note
  - GitHub project link
- Moved share text generation into a tested helper.
- Updated README and the launch page to show the share flow.

## How To Try It

Run Token Shredder, send a local usage event, then open backstage and use the share panel.

```bash
node examples/post-usage-node.mjs
```

The share panel can copy:

- detailed summary
- English launch post
- Chinese launch post
- share URL
- PNG share card

## Privacy Notes

- Share text contains cost estimates and token usage summary only.
- It does not include prompts, completions, messages, or API keys.
- Token Shredder remains local-first and does not upload usage data.

## Known Limits

- macOS is still the primary target.
- Builds are unsigned and not notarized.
- Windows and Linux need community testing.
- Full streaming usage extraction is not implemented.
- SDK / CLI wrappers are not implemented yet.
