# Token Shredder v0.1.17

A tiny local desktop pet that shreds your AI token spending in real time.

This update focuses on launch assets. Token Shredder now has a repeatable asset-generation workflow for README, GitHub Pages, release notes, and social posts.

## Highlights

- Added `npm run assets:launch`.
- Added `scripts/generate-launch-assets.mjs`.
- Generated launch assets from real local pet artwork:
  - social preview
  - getting-started card
  - troubleshooting card
  - skin showcase
  - static demo storyboard
  - animated SVG demo
- Updated GitHub Pages landing page to show first-run and troubleshooting visuals.
- Added `docs/ASSETS.md` with regeneration instructions and launch-asset rules.
- Updated README, Launch Kit, and Release Guide with the new asset workflow.
- Kept privacy / trademark boundaries explicit: no real company logos, no real currency images, no fake billing accuracy.

## How To Regenerate

```bash
npm run assets:launch
```

On macOS, the script uses Chrome / Chromium headless to export PNG and JPG files from generated SVG sources. If Chrome is unavailable, SVG assets are still generated.

## Privacy Notes

- The generated assets use local artwork and product copy only.
- No user usage logs, prompts, completions, messages, or API keys are included.
- The assets are launch cards, not untouched product screenshots.

## Known Limits

- The main README still uses the existing GIF for broad compatibility.
- Chrome / Chromium is required for PNG/JPG regeneration.
- The macOS build remains unsigned and not notarized.
