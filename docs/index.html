Token Shredder

A tiny desktop pet that shreds your AI token spending in real time. Point your agent at a local collector, then watch estimated cost turn into TOKEN blocks.

This is an early open-source project made by a non-professional developer with AI assistance. It works, but rough edges are expected. Feedback, privacy review, packaging help, and new skins are welcome.

Five tiny ways to feel token spend

Why people might actually use it

Real usage events

Send custom token usage or OpenAI-style usage to the local collector and the pet reacts to estimated cost.

Local privacy boundary

The app listens on 127.0.0.1, records usage numbers, and does not need to log prompts, completions, or API keys.

Funny enough to share

The joke is simple: stop pretending tokens are free. The useful part is that it works beyond the joke.

Try the local API

curl -X POST http://127.0.0.1:17391/usage \\
  -H "Content-Type: application/json" \\
  -d '{"source":"my-agent","inputTokens":120000,"outputTokens":45000}'
